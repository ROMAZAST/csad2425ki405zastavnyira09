## Repository Details
This repository contains a project for a Tic-Tac-Toe game created using Arduino for the CSAD course.

## Task Details
The main task is to develop a Tic-Tac-Toe game with the following features:

### Menu Options
- **Play Modes**:
  - **Man vs AI**
  - **Man vs Man**
  - **AI vs AI**
    - **Random Move**
    - **Win Strategy**
